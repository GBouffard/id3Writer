:cd: id3Writer :notes:
=================

Personal project made in Javascript to write i3d tags.

Objectives of exercise
----
To learn about Javascript, Jasmine and jQuery.

Technologies used
----
- Javascript
- Ndde
- Spotify API

Node Packages:
- recursive-readdir-synchronous
- node-id3
- fs
- request
- get-json

How to run it
----
```
git clone git@github.com:GBouffard/id3Writer.git
cd id3Writer
npm install
npm write-id3
```
